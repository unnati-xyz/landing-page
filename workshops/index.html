Data Science at Scale

The biggest problem in the field of Data Science is building platforms that scale. What we mean by this is, using machine learning models or cleaning data on a small dataset seems to work just fine. But what happens when you need to run the same models on 100's GBs of data? Things break and finally you start seeing the complexity in solving large problems. This workshop is aimed at building Data pipelines that can handle large amounts of data using Apache Spark and performing analytics at scale.

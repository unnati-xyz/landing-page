DATA SCIENCE AT SCALE


The biggest problem in the field of Data Science is building platforms that scale. You'll find that running the same Machine Learning models or even simple Data Cleansing tends to fail when dealing with data that spans 100s of GBs. Its at this scale that things start to get more complex. This workshop is aimed at building Data pipelines that can handle large amounts of data using Apache Spark and performing analytics at scale.
